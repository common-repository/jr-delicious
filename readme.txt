== Description ==

This Delicious plugin allows you to display your latest Delicious bookmarks as a widget on your blog!

Simply enable the plugin through your admin panel, edit the JR Delicious settings in the Settings --> JR Delicious menu and enable the widget through the Appearance --> Widgets menu! They will now be displayed on your blog!

== Installation ==

1. Upload `jr-delicious.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Delicious Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets menu.
